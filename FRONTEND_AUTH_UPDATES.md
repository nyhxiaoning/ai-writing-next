## 核心功能特性

### 🔐 认证流程
1. **登录流程**：邮箱/密码验证 → API 调用 → JWT 存储 → 重定向
2. **注册流程**：表单验证 → API 调用 → 自动登录 → 重定向
3. **重置密码**：邮箱验证 → 发送邮件 → 邮件链接 → 设置新密码

### 🌍 国际化支持
- 所有认证页面支持多语言
- 重定向时保持语言前缀

### 📱 响应式设计

## 状态管理集成

### Zustand Store 方法使用
- `login(email, password, rememberMe)` - 用户登录
- `register(name, email, password, confirmPassword)` - 用户注册
- `logout()` - 用户登出
- `requestPasswordReset(email, locale)` - 请求重置密码
- `confirmPasswordReset(token, password, confirmPassword)` - 确认重置密码
- `checkAuth()` - 检查认证状态

### 状态属性
- `user` - 当前用户信息
- `isAuthenticated` - 认证状态
- `isLoading` - 加载状态
